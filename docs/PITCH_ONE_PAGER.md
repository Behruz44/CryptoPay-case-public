# CryptoPay — One-Pager для партнёров

---

## Что предлагается

**Готовая кастодиальная крипто-инфраструктура** (14 микросервисов, NestJS/TypeScript + Flutter), покрывающая ~70% технических требований крипто-депозитария и ~60% крипто-магазина по стандартам НАПП.

| | |
|---|---|
| **Кошельки** | HD-деривация BIP-44, hot/cold sweep, изолированный signing-service |
| **Депозиты** | Реальный on-chain мониторинг USDT/Polygon, multi-RPC failover, DLQ + recovery |
| **Платежи** | Saga-оркестратор, идемпотентность, двойная запись, BigInt-точность |
| **KYC** | Sumsub-интеграция (webhooks + HMAC) |
| **Безопасность** | Внутренний аудит: 30 исправлений, отчёт прилагается |
| **Инфраструктура** | Docker/K8s/Helm/CI-CD/Prometheus — готово к развёртыванию |
| **Клиент** | Flutter-приложение: кошелёк, QR, история, профиль |

## Модели сотрудничества

1. **Asset sale** — полная передача кода, документации и прав + onboarding
2. **White-label лицензия** — код остаётся у нас, вы получаете право использования + поддержку (recurring)
3. **Техническое партнёрство** — совместная доработка под вашу лицензию с долей

## Что честно требует доработки

KMS/HSM-активация (модуль готов), внешний пентест, нагрузочное тестирование, AML-мониторинг транзакций. Оценка: 3–6 месяцев, 2–3 инженера. Полный список — в техническом досье.

## Правила outreach (важно)

1. **Сначала NDA, потом код.** Досье и отчёты — можно сразу; исходники — только после подписания NDA
2. **Не обещать то, чего нет.** Раздел «что требует доработки» показывать честно — это повышает доверие при due diligence
3. **Юрист до сделки.** Любой договор (продажа IP, лицензия) — только через юриста
4. **Фиксировать IP.** Убедиться, что git-история чистая и авторство доказуемо (уже так)
